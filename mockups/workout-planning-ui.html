How it works

  • Rest sits between sets as a visual separator — clear what it refers to
  • Each rest value is independently editable (e.g., shorter rest after warm-up, longer between working sets)
  • "+ Add Set" copies weight, reps, and rest from the last row — just click and go
  • No top-level "quick config" bar — the per-row approach avoids duplication
  • Warm-up sets show the percentage hint (60%) next to the weight
  • Click any value to edit inline — no dialogs needed
  • Click exercise header to collapse/expand
